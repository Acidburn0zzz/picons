Guide

You've got the following options on installing...

  1. Directly from your box, if you're using OpenVIX or OpenPLI. OpenVIX will have immediate updates, on OpenPLI you'll need to wait a bit longer.

  2. If you can't wait that long or you're using a different image, you can install the file "picons-feed.conf" in "/etc/opkg" on your box. Reboot and then install the picons from the menu.

  3. Manual install with the files from the Downloads section.
    The "exe" files, are self extracting 7z files to be used on Windows, you can also use 7zip program to extract them. They don't contain symlinks and will take up the most space.

The "tar.gz" files, are archives to be used on Linux or directly on your box. They contain symlinks and will take up less space.

The "ipk" files, are install packages to be used directly on your box. They also contain symlinks and are the same as the ones you install from the feed.
